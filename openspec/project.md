# Project Context

## Title
Spam/Ham Email Classifier

## Description
This project builds a logistic regression classifier to detect spam messages in SMS data. It includes preprocessing, training, evaluation, and a Streamlit-based interactive dashboard. The goal is to demonstrate a full ML pipeline using CLI tools and OpenSpec documentation.

## Tech Stack
- Python 3.9
- scikit-learn
- pandas
- Streamlit
- joblib
- matplotlib / seaborn
- OpenSpec CLI

## Conventions
- CLI scripts stored in `scripts/`
- Models saved in `models/`
- Visualizations saved in `reports/visualizations/`
- App entry point: `app/streamlit_app.py`
- Dataset path: `datasets/processed/sms_spam_clean.csv`
- OpenSpec proposals stored in `openspec/changes/`
